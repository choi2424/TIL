리액트 함수형 컴포넌트
 : 자바스크립트 함수문법을 이용
  함수이름은 대문자로 작성
   작은 단위의 ui(자바스크립트 프로그래밍)
 - 순수 자바스크립트로 작성 - 파일의 확장자 .js
   JSX문법이 포함되서 작성- 파일의 확장자 .jsx 로도 가능

함수형 컴포넌트로 만들고 그것을 태그처럼 사용한다
 
 -import Hello~~; 로 가져다 쓰면 내무적인 문법이 Hello.js 를 찾는다  Hello.js 가 없으면 Hello.jsx를 참조한다

 참고 > 리액트 클래스형 컴포넌트로 만들다보니 문제점이 많아서 현재는 전부 함수형 컴포넌트로 만든다

리턴 안에 작성된 문법은 html문법으로보이지만 JSX문법으로 작성되어 있다
예)
function Employee(props) {
    return (
        <div>
            <h1>사원번호 : {props.number}</h1>
            <h2>사원이름 : {props.name}</h2>
        </div>
    );
}

JSX : JavaScript + (HTML,XML)포함한 형태의 언어
      자바스크립트 표준문법은 아니다.
  함수형 컴포넌트 생성시 리턴형태로 사용,
  JSX -> (Babel 도구) -> ReactJS(리액트 자바스크립트 컴파일 변환) 

렌더링 : 컴포넌트가 호출이되어 , 그 결과가 웹브라우저 화면에 출력되는 것

useState(0) 구문이 동작되면, 2개요소를 갖는 배열형태의 기억장소가 생성된다.
[0] 첫번째 요소는 초기값을 갖게되는 기억장소가 되고, [1] 두번째 요소는 set 기능을 하는 함수가 존재하게 된다.
set 기능은 [0] 첫번째 요소의 기억장소의 값을 변경하는 목적으로 만들어져 있다.
